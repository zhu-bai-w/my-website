# 项目简介

《属于“你”的箱庭》是一个以记忆、关系与选择为核心的叙事模拟游戏项目。第一版目标是完成可游玩的 Demo，并把核心玩法、叙事结构和系统循环整理成稳定的策划文档。

## 我的职责

- 梳理项目主题、玩家目标和体验关键词。
- 设计核心系统循环、关卡节奏和叙事触发结构。
- 维护策划文档，推进任务拆解和版本复盘。

## 核心玩法

玩家在一个小型箱庭空间中探索线索、整理关系、做出选择，并通过行动影响角色记忆和场景状态。

### 体验目标

玩法不追求复杂操作，而强调“我在理解一个人、一段关系和一个空间”的过程。

## 系统设计

当前系统以探索、信息收集、关系变化和结局条件为主。后续会继续细化事件触发、状态变量和可视化反馈。

## 数值与流程

第一版采用轻量变量，不做过重数值。重点是让玩家能看见选择带来的关系变化和场景反馈。

## 团队协作

项目文档以飞书为主，代码与素材版本通过 Git 管理。每个阶段保留任务拆解、问题记录和复盘结论。

## 当前进度

Demo 正在开发中，主要工作集中在玩法闭环、叙事片段和交互原型。

## 复盘与收获

这个项目让我更明确地意识到：游戏策划不是写一个漂亮概念，而是把概念拆成玩家能操作、团队能实现、版本能验证的系统。
